# Licita Radar API

API desenvolvida em C# com ASP.NET Core para monitoramento e gerenciamento de licitações públicas.

## Objetivo

A Licita Radar API tem como objetivo:

- monitorar licitações públicas
- coletar editais automaticamente
- filtrar oportunidades
- gerar alertas
- integrar IA para resumo de editais

---

# Tecnologias

- ASP.NET Core
- Entity Framework Core
- PostgreSQL
- Swagger
- JWT Authentication
- Redis
- RabbitMQ
- Docker
